# CLAUDE.md — правила для этого проекта

## Язык интерфейса

**Весь пользовательский текст на сайте должен быть на русском языке.**

Исключения (оставлять как есть):
- Название продукта: **JagerVPN**
- Технические термины: **VPN**, **AES-256**, **Telegram**, **iOS**, **Android**, **Windows**, **macOS**, **Gbps**, **Uptime**
- Ссылки на внешние сервисы

## Структура проекта

- `lib/constants.ts` — центральный конфиг: tagline, навигация, список стран
- `components/sections/` — секции страницы: Hero, Features, Countries, Pricing, CTA
- `components/layout/` — Header, Footer
- `app/layout.tsx` — метаданные страницы, `lang="ru"`

## Правила при добавлении нового контента

1. Весь новый пользовательский текст пишется на **русском**.
2. Технические термины (протоколы, платформы, названия) оставлять на английском.
3. Атрибуты `aria-label` и `alt` — переводить на русский.
4. Метаданные (`description`, `keywords`) — на русском.
